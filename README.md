# 📱 Mundo realme - Site sobre Tecnologia

Um site institucional e interativo focado na marca de tecnologia **realme**, desenvolvido utilizando as tecnologias fundamentais da web: HTML, CSS e JavaScript. O projeto apresenta os principais pilares de inovação da marca com um design moderno, responsivo e interativo.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica de todo o conteúdo da página.
* **CSS3:** Estilização personalizada utilizando a identidade visual da realme (tons escuros e amarelo vibrante), além de técnicas de *Grid* e *Flexbox* para garantir total responsividade em dispositivos móveis.
* **JavaScript (ES6):** Manipulação do DOM para criar interatividade, como rolagem suave ao clicar em botões e eventos de clique nos cards de produtos.
